# Sprints

## Sprint 0

Objetivo: criar somente a base técnica inicial do repositório.

Inclui monorepo, frontend, backend, workers, packages compartilhados, documentação inicial, `.env.example`, `AGENTS.md`, healthcheck e página inicial simples.

Não inclui banco, autenticação, permissões, tenants, integrações, IA, WhatsApp, filas, jobs ou módulos operacionais.
